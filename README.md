# موقع لينام سكوتش - Linam Scotch Website

## نظرة عامة
موقع تفاعلي متكامل بصفحة واحدة (One-Page Website) لمشروع "لينام سكوتش" المتخصص في استيراد وبيع قطط السلالة الاسكتلندية (سكوتش) في منطقة الجوف.

## المميزات الرئيسية

### 1. التصميم المتجاوب (Responsive Design)
- يعمل بشكل مثالي على جميع الأجهزة (الهواتف الذكية، الأجهزة اللوحية، أجهزة الكمبيوتر)
- تصميم متوافق مع اللغة العربية (RTL - Right to Left)

### 2. الأقسام الرئيسية
- **الصفحة الرئيسية (Hero Section)**: عنوان ترحيبي مع مشغل فيديو تفاعلي
- **من نحن (About)**: معلومات عن المشروع مع صورة
- **خدماتنا (Services)**: 5 بطاقات خدمات تفاعلية
- **الإحصائيات (Statistics)**: رسوم بيانية لنتائج الاستبيان
- **الأسئلة الشائعة (FAQ)**: قوائم قابلة للطي (Accordion)
- **تواصل معنا (Contact)**: معلومات التواصل وأيقونات وسائل التواصل الاجتماعي

### 3. لوحة الألوان
```
اللون الأساسي (Primary): #8b84b7 (بنفسجي متوسط)
اللون الثانوي: #aea7c6 (بنفسجي فاتح)
اللون الثالث: #b2a8cc (بنفسجي فاتح جداً)
لون الخلفية المتبادل: #f1e3f0 (وردي فاتح جداً)
لون الخلفية الأساسي: #fcfbfd (أبيض مائل للبنفسجي الفاتح جداً)
```

### 4. التفاعلية والحركات
- تأثيرات Hover على الأزرار والصور والبطاقات
- تأثيرات Fade-in عند التمرير (Scroll Animation)
- قوائم FAQ قابلة للطي بسلاسة
- شريط تقدم متحرك للإحصائيات
- شريط التنقل ثابت (Sticky Navbar)

### 5. الخطوط (Typography)
- استخدام خطوط عربية حديثة: Cairo و Tajawal
- عناوين بحجم كبير وسميك (Bold)
- نصوص فرعية بحجم عادي ومريح للقراءة
- تأثير توهج خفيف على العناوين

## محتويات الملفات

### index.html
الملف الرئيسي يحتوي على:
- بنية HTML الكاملة
- جميع الأقسام والمحتوى
- روابط الخطوط والأيقونات
- عناصر الفيديو والصور

### styles.css
ملف التنسيق يحتوي على:
- متغيرات CSS للألوان والانتقالات
- تنسيق جميع الأقسام
- تأثيرات الحركة والانتقالات
- تصميم متجاوب للأجهزة المختلفة
- تأثيرات Hover والتفاعلية

### script.js
ملف JavaScript يحتوي على:
- وظائف التمرير السلس (Smooth Scrolling)
- وظائف FAQ Accordion
- تأثيرات الظهور عند التمرير (Scroll Animation)
- وظائف شريط التنقل الثابت
- تأثيرات شريط التقدم المتحرك
- وظائف التنقل النشطة

## كيفية التخصيص

### 1. استبدال الشعار
ابحث عن السطر التالي في `index.html`:
```html
<img src="https://via.placeholder.com/80x80?text=لينام+سكوتش" alt="لينام سكوتش" class="logo">
```
استبدل الرابط برابط شعارك الفعلي.

### 2. استبدال الفيديو
ابحث عن عنصر الفيديو في `index.html`:
```html
<video width="400" height="300" controls poster="...">
    <source src="..." type="video/mp4">
</video>
```
استبدل رابط الفيديو برابط الفيديو الفعلي.

### 3. استبدال الصور
استبدل جميع روابط الصور (placeholder) برابط الصور الفعلية:
```html
<img src="https://via.placeholder.com/..." alt="...">
```

### 4. تعديل معلومات التواصل
في قسم Contact، عدّل:
- رقم الهاتف
- روابط وسائل التواصل الاجتماعي
- الموقع الجغرافي

### 5. تعديل الألوان
في `styles.css`، عدّل متغيرات CSS:
```css
:root {
    --primary-color: #8b84b7;
    --secondary-color: #aea7c6;
    --tertiary-color: #b2a8cc;
    --light-bg: #f1e3f0;
    --main-bg: #fcfbfd;
}
```

## التثبيت والتشغيل

### الطريقة 1: استخدام Python
```bash
cd linam-scotch-website
python3 -m http.server 8000
```
ثم افتح المتصفح على: `http://localhost:8000`

### الطريقة 2: استخدام Live Server (VS Code)
1. ثبت تطبيق Live Server
2. انقر بزر الماوس الأيمن على `index.html`
3. اختر "Open with Live Server"

### الطريقة 3: رفع على خادم ويب
انسخ جميع الملفات إلى خادمك وافتح الموقع عبر المتصفح.
